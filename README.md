# Genetic algorithm
This is a high efficiency realization in c++ for the genetic algorithm.

# Dependencies

- [Eigen](https://eigen.tuxfamily.org/index.php?title=Main_Page)
- [Boost](https://www.boost.org/)

# Installation

```{bash}
 cd Genetic_algorithm
 autoreconf -i
 mkdir build
 cd build
 ../configure
 make
 make install
```
